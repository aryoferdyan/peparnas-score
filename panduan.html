Cara Mengedit Gambar, Nama, dan Skor Set

Mengedit Gambar

Untuk mengubah logo atau gambar tim:

  1. Klik pada gambar/logo yang ingin Anda ganti.
  2. Pilih gambar baru dari perangkat Anda melalui jendela unggah yang muncul. (FORMAT HARUS .PNG)
  3. Gambar akan otomatis diperbarui dengan yang baru.

Mengedit Nama Tim

Untuk mengganti nama tim:

  1. Klik pada nama tim yang ditampilkan di tabel skor.
  2. Nama tim akan menjadi dapat diedit.
  3. Ketikan nama tim baru dan tekan tekan di luar kotak untuk menyimpan perubahan.

Mengedit Skor Set

Untuk mengedit skor set pertandingan (skor tiap set):

  1. Klik pada skor set (misalnya, di kolom "I", "II", dll.) yang ingin diubah.
  2. Nilai skor akan menjadi dapat diedit.
  3. Masukkan skor yang diinginkan untuk masing-masing tim dan tekan di luar kotak untuk menyimpan perubahan.
